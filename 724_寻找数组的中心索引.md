## 代码
```C++
class Solution {
public:
    int pivotIndex(vector<int>& nums) {
        int len = nums.size();
        int temp = 0;
        int sum = accumulate(nums.begin(),nums.end(),0);
        for(int i = 0;i < len;i++){
            if(temp * 2 + nums[i] == sum){
                return i;
            }else{
                temp += nums[i];
            }
        }
        return -1;
    }
};
```
## 思路

原本以为可以对数组进行变换来搜有没有中心索引，还在想需要深搜什么的。后来发现数组是不变的。那就直接遍历找一下就可以了。用了一下`stl`中的`accumulate`函数来计算和。
