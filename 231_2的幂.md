## 题目
给定一个整数，编写一个函数来判断它是否是 2 的幂次方。

**示例1**
```
输入: 1
输出: true
解释: 20 = 1
```

**示例2**
```
输入: 16
输出: true
解释: 24 = 16
```

**示例3**
```
输入: 218
输出: false
```

## 代码（基本做法)
```C++
class Solution {
public:
    bool isPowerOfTwo(int n) {
        if(n <= 0){
            return false;
        }
        while(n % 2 == 0){
            n >>= 1;
        }
        return n == 1;
    }
};
```

## 代码（位运算：去除二进制中最右边的 1)
```C++
class Solution {
public:
    bool isPowerOfTwo(int n) {
        if(n <= 0) return false;
        long x = n;
        return (x & (x - 1)) == 0;
    }
};
```
